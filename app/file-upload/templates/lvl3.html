Уровень 3: Удаление файла логов с помощью Path Traversal

Условие

Сервер хранит файл логов, который нужно удалить, чтобы скрыть следы. Используй первый уровень, чтобы найти путь к файлу логов (подсказка: он находится в /app/system/serverfiles/logs/). Сервер принимает только файлы с расширением .jpg, и они должны быть валидными изображениями (по магическим байтам). Используй Path Traversal, чтобы сохранить файл в нужной папке. Замаскируй Python-код под .jpg файл, чтобы удалить файл логов. Уровень будет считаться пройденным, если в папке логов останется только один .jpg файл.

Подсказка по Path Traversal: Базовый путь для сохранения — /app/static/uploads/. Чтобы сохранить файл в /app/system/serverfiles/logs/, используй относительный путь, например: ../../system/serverfiles/logs/exploit.jpg.
