SQL Injection

SQL-инъекция позволяет атакующему выполнить произвольные SQL-запросы, обходя авторизацию или извлекая данные из базы.

Подробнее

Cross-Site Scripting (XSS)

XSS позволяет вставить вредоносный JavaScript-код на страницу, который выполнится в браузере жертвы.

Подробнее

Cross-Site Request Forgery (CSRF)

CSRF заставляет жертву выполнить нежелательные действия на сайте, где она аутентифицирована.

Подробнее

File Upload Vulnerabilities

Уязвимости загрузки файлов позволяют атакующему загрузить вредоносный файл и выполнить его на сервере.
